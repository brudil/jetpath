

Standard:

```js
<Toast title="Example toast" message="This toast is essentially just an alert" />
```

with actions:

```js
<Toast title="Example toast" message="This toast is essentially just an alert" actions={[
  {
    title: 'Publish',
    type: 'action'
  },
  {
    title: 'Keep draft',
    type: 'dull'
  }
]} />
```
